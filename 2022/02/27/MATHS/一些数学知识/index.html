    6. 仿射函数和线性函数

    仿射函数和线性函数的区别

    仿射函数即由由1阶多项式构成的函数，一般形式为 $ f (x)= A x + b$ ，这里，$A$ 是一个 $m * k $ 矩阵，$x$ 是一个 $k$ 向量, $b$是一个 $m$ 向量，实际上反映了一种从 k 维到 m 维的空间映射关系。

    设$f$是一个矢性（值）函数，若它可以表示为 $f(x_1,x_2,…,x_n)=A_1x_1+A_2x_2+…+A_nx_n+b$，其中 $A_i$ 可以是标量，也可以是矩阵，则称f是仿射函数。

    其中的特例是，标性（值）函数 $f(x)=ax+b$，其中 $a、x、b$ 都是标量。此时严格讲，只有 $b=0$ 时，仿射函数才可以叫“线性函数”（“正比例”关系）。

    III. 凸优化

    求解一个一般性的最优化问题的全局极小值是非常困难的，至少要面临的问题是：函数可能有多个局部极值点，另外还有鞍点问题。对于第一个问题，我们找到了一个梯度为 0 的点，它是极值点，但不是全局极值，如果一个问题有多个局部极值，则我们要把所有局部极值找出来，然后比较，得到全局极值，这非常困难，而且计算成本相当高。第二个问题更严重，我们找到了梯度为 0 的点，但它连局部极值都不是，典型的是 $X^3$ 这个函数

    梯度下降法和牛顿法等基于导数作为判据的优化算法，找到的都导数/梯度为0的点，而梯度等于0只是取得极值的必要条件而不是充分条件。如果我们将这个必要条件变成充分条件，即：

    问题将会得到简化。如果对问题加以限定，是可以保证上面这个条件成立的。其中的一种限制方案是：

    对于目标函数，我们限定是凸函数；对于优化变量的可行域（注意，还要包括目标函数定义域的约束），我们限定它是凸集。

    同时满足这两个限制条件的最优化问题称为凸优化问题，这类问题有一个非常好性质，那就是局部最优解一定是全局最优解。接下来我们先介绍凸集和凸函数的概念。

    概率论

    基本概念

    1. $A^c$ S’interprète comme “non A”

    2. 条件概率：$P(A|B) = \frac{P(A \and B)}{P(B)}$

    3. 全概率公式 ( théorème des prebabilité totales )

      $P(A) = \sum_{i=1}^{n}P(A|B _i)P(B_i)$

    4. 贝叶斯公式 ( Formules de Bayes )

      在已知事件 A 发生的条件下，寻找导致 A 发生的各种原因 $B_i$ 的概率

      $P(A|B) = \frac{P(A|B)P(A)}{P(B)}$

      $P(B_i|A) = \frac{P(A|B_i)P(B_i)}{\sum _{j=1}^{n}P(A|B_j)P(B_j)}$

    5. 事件独立性 ssi

      $P(A \and B) = P(A)P(B)$

      若 A 和 B 相互独立，则 $P(A|B) = P(A)$

    6. $P_X(B) = P(X \in B) = P(X^{-1}(B))$

    分布函数( fonction de répartition ) & 概率密度函数( la densité de probabilité )

    分布函数

    1. 概念

    $F_X(x) = P_x ( \ ]-\infty, x[ \ ) = P(X \in \ ] -\infty,x \ ]) = P(X \leq x), x \in R$

    2. 性质
    1. 非递减
    2. $lim_{x \rightarrow +\infty} F_X(x) = 1$
    3. $lim_{x \rightarrow -\infty} F_X(x) = 0$
    4. $P(a < X \leq b) = F_X(b) - F_X(a)$
    5. $F_X$ est continue à droite
    6. $P(X = x) = F_X(X^+) - F_X(X^-)$
    3. 分位点( quantile )

    ​ $a_r = inf \left{x; F_X(x) \geq r\right}, \ a_r \in R, \ r \in ]0,1[$

    4. 几个性质 ( moments )

    1. 期望 ( la moyenne )

      1. $E(x) = \sum x P(X=x) = \int xf_X(x) \ dx$

      2. $E(aX+b) =aE(X) +b$

      3. Une variable aléatoire est dite centrée si $ E(X) = 0$. Elle est dite centrée et réduite si de plus $\sigma (X) = 1$.

    2. 方差 ( variance ) 也写作 $V(X)$

      1. $D(X) =E(X^2) - E^2(X) = E((X-E(X))^2)$
      2. $D(aX+b) = a^2 \ D(X)$

    3. 标准差 $\sigma$ ( l’écart-type )

      1. $\sigma(X) = \sqrt{D(X)}$

      2. 标准化随机变量

        $Z = \frac{X-E(X)}{\sigma(X)}$

        $E(Z) =0, \ D(Z) =1$

    4. 一些理解和矩生成函数 ( la fonction génératrice des moments )

      矩是 $X$ 的期望值，例如$E(X), E(X^2), E(X^3)$等。 第一矩是$E(X)$， 第二矩是$E(X^2)$， 第三矩是$E(X^3)$， … 第 $n$ 个矩是$E(X^n)$， 期望是平均值，方差是分布的分布程度。但是，必须有其他的特征来定义分布。例如，第三矩是分布的不对称性。第四关是尾巴有多沉、多厚。

      4.1 La fonction génératrice des moments

      Def. $G_X(t) = E(e^{tX}) = \sum e^{tx}P(X=x) = \int e^{tx} \ f_X(x) \ dx $

      ​ On note alors que $G_X(t) = L(f_X)(-t), \ L 是拉普拉斯变换$

      推论：

      ​ $G_X^{(k)}(0) = E(X^k)$

      4.2 La fonction caractéristique

      Def. $\phi _X(t) = E(e^{itX}) \ : \ R \rightarrow C$

      ​ $\phi _X(t) =\sum e^{itx}P(X=x) = \int \ e^{itx}f_X(x) \ dx = F(f_X)(-t), F 表示傅里叶变换 $

      推论

    5. 重要不等式

    1. Inégalité de Markov

      pour tout $a>0$ , on a $P(|X|\geq a) \leq \frac{E(|x|)}{a}$

      推论：

      Si $ f : I \in R \rightarrow R$ est une fonction mesurable strictement croissante et non négative,

      et si $Y$ est une v.a. telle que$ P (Y \in I) = 1$,

      alors pour tout $a \in I$ ,

      on a $P (Y \geq a) \leq \frac{E(f(Y ))}{f(a)}$

    2. Inégalité de Chebychev

      Soit X une v.a. du second ordre. Pour tout $ \epsilon > 0$,

      on a alors $P (|X| \geq\epsilon) \leq \frac{E(X^2)}{\epsilon ^2}$
      On en déduit notamment que pour tout $\epsilon > 0$,
      on a $P (|X ( E(X)| \geq \epsilon) \leq \frac{V(X)}{\epsilon ^2} $

    3. Inégalité de Jensen

      Soit X une v.a. d’espérance finie.

      Soit $\phi : R \rightarrow R $ une fonction convexe.

      Alors $\phi(E(X)) \leq E(\phi(X))$

    4. Inégalité de Cauchy-Schwarz

      Soient $X$ et $Y$ deux v.a. réelles du second ordre.

      Alors $E(XY )$ existe et $E(|XY |) \leq E(X^2)^{\frac{1}{2}}E(Y^2)^{\frac{1}{2}}$

    5. Inégalité de Hölder

      Soient $X$ et$ Y$ deux v.a. réelles respectivement d’ordre $p$ et $q$,

      avec $1 < p, q < +\infty$ et $1/p+1/q= 1$.

      Alors, $XY$ est dans $L^1$ et $E(|XY |) \leq E(|X|^p)^{1/p} E(|Y |^q) ^{1/q}$

    中心极限定理 Central Limit Theorem

    中心思想是无论是什么分布的数据，当我们从中抽取相互独立的随机样本，且采集的样本足够多时，样本均值的分布将收敛于正态分布。

    从均值为$\mu$ , 方差为 $\sigma^2$ 的任意一个总体中抽取样本量为 $n$ 的样本，当 $n$ 充分大的时候，样本均值 $\bar X$ 近似服从均值为 $\mu $ , 方差为 $\frac{ \sigma^2 }{n}$ 的正态分布，那么$Z_n = \frac{\bar X -\mu}{\frac{\sigma }{\sqrt{n}}} \sim N(0,1)$

    大数定律

    大数定律告诉我们能用频率近似代替概率；能用样本均值近似代替总体均值。

    弱大数定律（依概率收敛）

    依概率收敛：当n越来越大时，随机变量x落在 $(c - \epsilon, c+ \epsilon ,\ \epsilon 为任意给定正数 )$ 外的概率趋近于0，还是有可能落在外面的，只不过可能是很小，且会随着n的增大，这种可能越来越小。

    随着n的增大，平均值接近真实期望值的可能性也在增大。

    强大数定律（几乎必然收敛）

    假定$X_n $为独立同分布的随机序列，且$E(X_1) = \mu, \ Var(X_1) = \sigma^2$ 存在，则样本均值 $\bar X = \frac{1}{n} \sum_{i=1}^n x_i\rightarrow \mu$

    随着n的增大，平均值基本上就接近真实期望值了

    伯努利大数定理：揭示了概率与频率的关系

    当N很大的时候，事件A发生的概率等于A发生的频率。

    辛利大数定理：揭示了算术平均值和数学期望的关系

    用算术平均值来近似实际真值是合理的。

    切比雪夫大数定律：揭示了样本均值和真实期望的关系

    $X_1,X_2,….,X_n$ 是相互独立的随机变量，具有相同的期望 $\mu$, 相同的方差 $\sigma^2$, 那么 $\frac{1}{n} \sum_{i=1}^n X_i \rightarrow \mu(依概率)$

    统计

    1. 最大似然估计（MLE）

    就是利用已知的样本结果信息，反推最具有可能（最大概率）导致这些样本结果出现的模型参数值！

    1. 写出似然函数$L_x(\theta)$
      1. 连续：$L_x(\theta_1…\theta_k) = \prod_{i=1}^n f(x_i ;\theta_1,…\theta_k)$
      2. 离散：$L = \prod_{i=1}^n P(X_i = x_i)$
    2. 有必要的时候写出 $ln(L)$
    3. 对 $\theta$ 求导=0，解出$\theta$；若不可能为 0，那就是单调递增递减
    4. 本质上求使得似然函数最大的 theta

    2. 贝叶斯最大后验概率估计(MAP)

    2.1 贝叶斯公式

    $P(A|B) = \frac{P(B|A)}{P(B)} P(A)$

    $P(A|B):后验概率；P(B|A):似然函数；P(A):先验概率；$

    $ \frac{P(B|A)}{P(B)} : 标准似然比$

    • 如果标准似然比>1，则先验概率P(A)得到增强，事件P(B)的发生会增大事件AA发生的可能性；(B 发生的情况下 A 发生的概率比单纯 A 发生的概率要高)
    • 如果标准似然比=1，则先验概率P(A)保持不变，事件P(B)的发生不影响事件AA发生的可能性；
    • 如果标准似然比<1，则先验概率P(A)得到削弱，事件P(B)的发生会降低事件P(A)发生的可能性。

    由贝叶斯公式和全概率公式可得：

    $P(A_i|B) = \frac{P(B|A_i)P(A_i)}{P(B)} = \frac{P(B|A_i)P(A_i)}{\sum P(B|A_i)P(A_i)}$

    2.2 最大后验（MAP）：

    最优的theta 不仅仅来自于数据，也来自于我们事先知道的知识（是怎样的 分布）

    $ \mathop{argmax}\limits_{\theta} \ P(θ|X)=\mathop{argmax}\limits_{\theta} \frac{P(X|θ)P(θ)}{P(X)} ∝ \mathop{argmax}\limits_{\theta} \ P(X|θ)P(θ)$

    $P(X|\theta) 是似然函数$

    $P(\theta) 是先验概率，\theta 的分布我们事先假设$

    $由于X的先验分布P(X)是固定的，可通过分析数据获得，其实我们也不关心XX的分布$
