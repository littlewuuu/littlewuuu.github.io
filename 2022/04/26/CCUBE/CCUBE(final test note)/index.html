chapitre 2: les voir autrement

2.1 un signal fontamental : la sinusoïde complexe

2.2 transformation de Fourier

def. La transformation de Fourier

$F_{cc}x(f) = \int_{-\infty}^{+\infty}x(t)e^{-j \ 2\pi ft} \ dt$

def. La transformation de Fourier inverse

$x(t) = \int_{-\infty}^{+\infty}F_{cc} \ x(f) \ e^{j \ 2\pi ft} \ df$

2.3 transformation de Laplace bilatère

def.

$Lx(s) = \int_{-\infty}^{+\infty} \ x(t)e^{-st} \ dt$

$s_0 $ est un pôle si $lim_{s \rightarrow s_0 \ Lx(s) \ = \ \infty}$

2.4 transformation Fourier (temps discret)

def.

$F_{dc}x(\lambda) \ = \ \sum_{n = -\infty}^{n =+\infty}x[n]e^{-j2\pi \lambda n}$

2.5 transformation en z bilatère

def.

$Zx(z) = \sum_{n = -\infty}^{+\infty}x[n]z^{-n}$

$s_0 $ est un pôle si $lim_{s \rightarrow s_0 \ Zx(z) \ = \ \infty}$
