Séquence 1: signaux et systèmes.

chapitre 1: concepts et outils fondamentaux

1.1 temps continu, temps discret

1.2 voyange dans le temps

  1. le signal renversé: $t \rightarrow x(-t) $ et $(x[-n])_n$
  2. le signal translaté: $t \rightarrow x(t-t_0)$
  3. le chargement d’échelle: $t \rightarrow x(\frac{t}{a})$

1.3 fonctions et suites élémentaires

1.4 impulsion et peigne impulsionnel

1.5 convolution

def.

$(xy)(t) = \int_{-\infty}^{+\infty} \ x(\tau)y(t-\tau) \ d\tau$ et $(xy)[n] = \sum_{k=-\infty}^{+\infty} \ x[k]y[n-k]$

性质

$(xy)z = x(yz)$

$x*\delta=x$

1.6 causalité, anticausalité

Une suit est dite causale si elle est nulle pour $n<0$

Elle est dite anticausale si elle nulle pour $n>0$

1.7 sinus cardial, noyau de Dirichlet

1.8 quatre formules fondamentales

chapitre 2: les voir autrement

2.1 un signal fontamental : la sinusoïde complexe

2.2 transformation de Fourier

def. La transformation de Fourier

$F_{cc}x(f) = \int_{-\infty}^{+\infty}x(t)e^{-j \ 2\pi ft} \ dt$

def. La transformation de Fourier inverse

$x(t) = \int_{-\infty}^{+\infty}F_{cc} \ x(f) \ e^{j \ 2\pi ft} \ df$

2.3 transformation de Laplace bilatère

def.

$Lx(s) = \int_{-\infty}^{+\infty} \ x(t)e^{-st} \ dt$

$s_0 $ est un pôle si $lim_{s \rightarrow s_0 \ Lx(s) \ = \ \infty}$

2.4 transformation Fourier (temps discret)

def.

$F_{dc}x(\lambda) \ = \ \sum_{n = -\infty}^{n =+\infty}x[n]e^{-j2\pi \lambda n}$

2.5 transformation en z bilatère

def.

$Zx(z) = \sum_{n = -\infty}^{+\infty}x[n]z^{-n}$

$s_0 $ est un pôle si $lim_{s \rightarrow s_0 \ Zx(z) \ = \ \infty}$

2.6 propriétés des transformations

2.7 quelques transformées

2.8 échantillonnage

$T_s$ est la période d’échantillonnage

$f_s = \frac{1}{T_s}$ est la fréquence d’échantillonnage

def.

$x_s[n] = x(nT_s)$

左下角的 s 表示采样后的
