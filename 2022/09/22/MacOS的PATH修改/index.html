MacOS PATH

环境变量相关的文件

按下述顺序加载：

  1. /etc/profile
  2. /etc/paths
  3. /etc/bashrc
  4. ~/.bash_profile 或 ~/.bashrc
  5. ~/.bash_login

/etc/profile, /etc/paths, /etc/bashrc 是系统级别配置文件，系统启动就会加载。

后面几个是当前用户级的环境变量，按照从前往后的顺序读取，如果 ~/.bash_profile文件存在，则后面的几个文件就会被忽略不读，如果~/.bash_profile文件不存在，才会以此类推读取后面的文件。

~/.bashrc没有上述规则，它是bash shell打开的时候载入的。
